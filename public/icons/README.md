# PWA Icons

This directory should contain the following icon files:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

To generate these icons:
1. Create a 512x512 PNG icon with your app logo
2. Use an icon generator tool like https://realfavicongenerator.net/
3. Or use ImageMagick: `convert icon-512x512.png -resize 192x192 icon-192x192.png`

For now, placeholder icons will be used during development.
